### 小宇宙FM获取播客节目数据

非常简单的一个 python 程序，只需要修改代码文件中第 8 行 `PID` 即可，请看下图示例。

![example_pid](example_pid.png)

能够获取到的数据有：
* `pid`：节目识别号
* `playCount`：播放量
* `duration`：节目时长
* `commentCount`：评论数
* `clapCount`：点赞数
* `favoriteCount`：标记喜欢
* `pubDate`：上传时间
